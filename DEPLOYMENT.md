# WebAppBazaar.com Deployment Guide

## Quick Start for Kerala Election Run Game

### Step 1: Prepare Your Files
Your game folder should contain:
```
dino-runner/
└── index.html
```

That's it! Just one file needed.

### Step 2: Access WebAppBazaar.com
1. Go to https://webappbazaar.com
2. Sign up or log in to your account

### Step 3: Create New Application
1. Click on "Create App" or "New Application"
2. Choose application name: e.g., "Kerala Election Run"
3. Select application type: "Static Website" or "HTML5 App"

### Step 4: Upload Your File
1. Upload `index.html` to your application
2. Make sure it's set as the **main/entry file** or **index file**
3. No other files needed!

### Step 5: Deploy
1. Click "Deploy" or "Publish"
2. Wait for deployment to complete
3. You'll get a URL like: `https://webappbazaar.com/[your-app-name]`

### Step 6: Share & Play
- Share your game URL with friends
- Start playing!

## Troubleshooting

### "Page not found" or blank page
- Make sure `index.html` is set as the main entry point
- Check that the file was uploaded successfully
- Refresh the page (Ctrl+F5 or Cmd+Shift+R)

### No sound effects
- Check browser audio settings
- Make sure sound is not muted on your device
- Some browsers require user interaction before playing audio (click/tap first)

### Game runs slowly
- Close other browser tabs
- Check internet connection
- Try a different browser
- Game is optimized for 60 FPS

### Party logos not showing
- Don't worry! The party logos are built-in colored circles
- No image files needed
- They will display automatically
